## 📦 安装

```bash
npm install @antv/l7-editor
```

## 🔨 使用

```tsx | pure
// ----js-----
import { L7Editor } from '@antv/l7-editor';

export default () => {
  return (
    <div>
      <L7Editor />
    </div>
  );
};
```

## ⌨️ 本地开发

```bash
# 安装依赖
npm install

# 运行页面
npm run start

# 打包
npm run build
```
